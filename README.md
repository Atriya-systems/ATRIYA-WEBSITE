# ATRIYA Systems | The Constitutional AI Substrate

**ATRIYA** is a high-fidelity, institutional-grade AI governance platform, designed as the primary execution substrate for provable AI decisions. This repository contains the source code for the official v3.0 corporate platform.

## Architecture
This front-end platform is built to reflect the high-density technical requirements of sovereign clients, eliminating whitespace and enforcing visual accountability across:
- **Platform Lattices**: The Core Connectivity and Governance architecture.
- **AcademicOS (Education)**: The sovereign learning substrate.
- **Enterprise Solutions / Legal / Government**: Vertical-specific adaptation nodes.

## Technology Stack
- **Framework:** Next.js 14+ 
- **Styling:** Tailwind CSS (Strict usage of Institutional Minimalist System - Indigo / Slate / White)
- **Animation Layer:** Framer Motion

## Core Institutional Principles
1. **Zero Leakage**: Strict scrubbing of all internal engineering tags (e.g., UEF- prefixes, AstraCore, ABOE, NodeDB) before commits.
2. **Technical Density**: Maximized engine lattice grids (up to 128 nodes) reflecting the true scale of the execution engine.
3. **Patent Security**: Enforced usage of public layer markers (`S-11`, `P-04`, `L-03`, `M-01`).

## Setup and Local Auditing
First, run the development server to audit the interface:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to visualize the execution environment.

## Deployment
This repository is optimized for Vercel deployment. Commits pushed to the `main` branch will automatically trigger strict build-time lint checks before public release.
